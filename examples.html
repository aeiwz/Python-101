Pandas GroupBy

Quick aggregation on a small DataFrame.

import pandas as pd

df = pd.DataFrame({
    "team": ["A","A","B","B","B","C"],
    "score": [10, 15, 7, 12, 9, 20]
})
print(df)
print("\\nMean by team:")
print(df.groupby("team")["score"].mean())

Matplotlib Plot

Simple sine wave plot.

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2*np.pi, 100)
y = np.sin(x)
plt.plot(x, y)
plt.title("Sine wave")
plt.xlabel("x")
plt.ylabel("sin(x)")
plt.show()
